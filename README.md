# Proyecto de Computación Gráfica e Interacción Humano-Computadora

## Representación Virtual del Puente de la Facultad de Ingeniería

Este proyecto consiste en el desarrollo de un entorno virtual interactivo del puente de la Facultad de Ingeniería de la UNAM, con el objetivo de representar el espacio arquitectónico y simular su uso durante eventos académicos, como ferias de empleo, exposiciones o actividades universitarias.

El escenario principal fue modelado en Blender y posteriormente integrado en OpenGL mediante la carga de modelos en formato `.obj`. El proyecto permite visualizar el puente, sus elementos estructurales, texturas, barandales, columnas, piso, techo y otros componentes característicos del espacio real.

---

## Objetivo del proyecto

El objetivo principal es crear una representación virtual navegable del puente de la Facultad de Ingeniería, incorporando elementos de computación gráfica como:

- Modelado 3D.
- Texturizado.
- Iluminación.
- Carga de modelos externos.
- Transformaciones geométricas.
- Animaciones.
- Interacción con el usuario.
- Navegación mediante cámara.

Además, el entorno busca representar cómo podría utilizarse el puente durante un evento académico mediante la colocación de stands, flujo de personas e información visual interactiva.
